h1 até h6

As tags h1 até h6 são títulos ou palavras/frases utilizadas para explicitar algo. Elas possuem tamanho maior também em negrito. Conforme aumenta o valor do h, as letras diminuem, de forma inversamente proporcional. Esta tag pode ser observada nesta página, por exemplo, no título: Desafio Resumo de Tags HTML Aprendidas.

p

A tag p é utilizada para iniciar um novo parágrafo.

(Voltar)

mark

A tag mark é utilizada para marcar uma palavra ou frase de forma similar ao uso de um marca-texto.

(Voltar)

small

A tag small é utilizada para exibir um texto em tamanho menor em relação ao texto ao redor.

(Voltar)

i

A tag i é utilizada para deixar a palavra/texto, em que está a tag, em itálico

(Voltar)

u

A tag u é utilizada para sublinhar a palavra/texto em que está a tag.

(Voltar)

a

A tag a serve para realizar a ligação à uma página interna do site ou externa ao clicar em alguma palavra selecionada, como por exemplo: youtube.com. Ou seja, ao clicar em "youtube.com" o usuário é redirecionado para o site do Youtube.

Existem alguns atributos para esta tag, como o utilizado, Target: que faz com que ao clicar, abra-se uma nova aba para abrir o link, ao invés de ser aberto na mesma página.

Outra funcionalidade é a de ancoragem, que possibilita a navegação entre as sessões do site, como pode-se ver no Índice desta página, que ao clicar na sessão desejada, vai para a respectiva sessão dentro do site.

(Voltar)

strong

A tag strong deixa a palavra/texto em negrito.

(Voltar)

ol

A tag ol é utilizada para fazer uma lista ordenada, no caso, em ordem numérica.

  1. exemplo-1
  2. exemplo-2
  3. exemplo-3

(Voltar)

ul

A tag ul é utilizada para fazer uma lista desordenada, no caso, organizando-a com pontos.

  • exemplo-1
  • exemplo-2
  • exemplo-3
(Voltar)

li

A tag li é utilizada para adicionar uma linha na lista, como pode-se ver nas listas acima em ol e em ul

(Voltar)

hr

A tag hr é basicamente uma linha fina que ocupa toda a página horizontalmente para melhorar a visibilidade de certos pontos.

(Voltar)

sub

A tag sub faz com que a palavra/texto selecionada(o) fique subscrita e é geralmente utilizada para fórmulas químicas, como (H2O), fórmulas matemáticas (10n) e notas de rodapé, como em (nota de rodapé1).

(Voltar)

sup

De forma oposta ao sub, a tag sup faz com que a palavra/texto selecionada(o) fique sobrescrita e é geralmente utilizada para expressões matemáticas, como (10n) e notas de rodapé, como em (nota de rodapé1).

(Voltar)

blockquote

A tag blockquote dá um espaçamento maior que um início de parágrafo. É geralmente utilizada para fazer citações de autores.

(Voltar)

font

Antes de mais nada, é importante dizer que a tag font caíram em desuso e não é mais suportada pelo HTML 5, suas formatações são feitas através do CSS. Com ela é possível alterar formatação, cor e tamanho da palavra/texto selecionada(o)

(Voltar)

del

A tag del é utilizada para mostrar que esta parte do documento foi deletada: Esse texto foi deletado.

(Voltar)

abbr

a tag abbr é utilizada para representar abreviações. Com o atributo tittle se torna possível mostrar ao usuário o que significa essa abreviação passando o mouse por site da sigla.HTML

(Voltar)
